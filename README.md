# url-guess

> Finds random short URLs

Read an article about how unsecure URL shorteners are. Wanted to see how easy it
was to bruteforce find short URLs. _It's really easy._

## Install

```bash
$ npm install -g url-guess
```

## Use

```bash
$ url-guess # -h for help
```

## Automatically open

```bash
$ open $(url-guess) # Mac
```
